# ArchFlow — Construction Project Management Platform

A full-stack construction project management platform built with the MERN stack, designed to streamline communication and workflow between construction companies, managers, staff, and clients.

## Live Demo
🔗 [ArchFlow Live](https://arch-flow.vercel.app)

## Backend API
🔗 https://archflow-backend-ek0c.onrender.com

## Features

- **Multi-role Dashboard** — Separate dashboards for Admin, Manager, Staff, and Client
- **Project Management** — Task assignment, scheduling, and milestone tracking
- **Payment Tracking** — Payment requests, approvals, and receipt generation
- **Document Management** — Upload, manage, and share project documents
- **AI-Powered Chatbot** — Built-in assistant using Google Gemini API
- **Real-time Client Updates** — Live project progress notifications
- **Complaint Management** — Structured complaint submission and resolution
- **Monthly Reports** — Automated report generation for project progress

## Tech Stack

- **Frontend:** React.js, Vite
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **AI:** Google Gemini API
- **Authentication:** JWT
